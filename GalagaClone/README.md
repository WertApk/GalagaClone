# Galaga Clone

Bu proje, YZM104 Programlama II dersi kapsamında C++ ve SFML kütüphanesi kullanılarak geliştirilmiş basit bir Galaga tarzı 2D uzay oyunudur.

## Kullanılan Teknolojiler

- C++
- SFML
- Visual Studio
- Git ve GitHub

## Oyun Amacı

Oyuncunun amacı ekrandaki düşmanları vurarak seviyeleri tamamlamaktır. Düşmanlar hareket eder ve oyuncuya ateş eder. Oyuncunun canı sıfıra düşerse oyun biter. Tüm seviyeler tamamlanırsa oyuncu oyunu kazanır.

## Kontroller

- A: Sola hareket
- D: Sağa hareket
- Space: Ateş etme
- Yukarı/Aşağı Ok: Menüde seçim yapma
- Enter: Menü seçeneğini onaylama
- ESC: Menüye dönme
- R: Game Over veya You Win ekranında yeniden başlatma

## Oyun Özellikleri

- Ana menü sistemi
- PLAY, KEYS, GAMEPLAY ve QUIT seçenekleri
- Oyuncu hareketi
- Çoklu mermi sistemi
- Düşman hareket sistemi
- Sinüs fonksiyonu ile dalgalı düşman hareketi
- Rastgele düşman ateşi
- Çarpışma kontrolü
- Skor sistemi
- Can sistemi
- Level 1, Level 2 ve Level 3
- Game Over ekranı
- You Win ekranı
- Yeniden başlatma sistemi

## Derleme ve Çalıştırma

Proje Visual Studio ortamında C++ ve SFML kütüphanesi kullanılarak geliştirilmiştir.

Projeyi çalıştırmak için:

1. Visual Studio ile proje dosyası açılır.
2. SFML kütüphanesinin projeye bağlı olduğundan emin olunur.
3. `main.cpp` dosyası derlenir.
4. Program çalıştırılır.
